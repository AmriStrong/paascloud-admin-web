# paascloud-login-web

> paascloud-master 配套的运营平台前端项目

## paascloud主页

```
http://www.paascloud.net
```
## 运行项目

运行环境 node 6.x, npm 4.x

``` bash
npm install
npm run dev
```

## 项目展示

```
http://admin.paascloud.net/
```
